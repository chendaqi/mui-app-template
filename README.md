# mui-app-template

基于 webpack5 的 mui 应用模版，让您开发更便捷，告别低级语法，让项目自动化、模块化，让您开发起来得心应手，支持 es6+、scss 等语法。

## 使用

```bash
$ git clone https://github.com/xh-shen/mui-app-template.git projectName
```
